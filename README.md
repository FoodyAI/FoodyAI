##  Overview

Foody is an intelligent food analysis and health tracking application built with Flutter. It leverages advanced AI technology to analyze food images and provide detailed nutritional information. The app helps users make informed dietary choices by:

- üì∏ Taking or selecting food photos for instant nutritional analysis
- üîç Identifying food items and their nutritional content using AI
- üìä Tracking daily calorie intake and nutritional goals
- üìà Monitoring macronutrients (protein, carbs, fat)
- ‚≠ê Providing health scores for food choices
- üì± Offering an intuitive, modern interface for health management

## üöÄ Getting Started

### Prerequisites

- Flutter SDK (>=2.18.0)
- Dart SDK
- Android Studio / Xcode
- VS Code (recommended)

### Installation

1. **Clone the repository**
   ```bash
   git clone [repository-url]
   cd foody
   ```

2. **Install dependencies**
   ```bash
   flutter pub get
   ```

3. **Configure environment**
   - Create `.env` file in root directory
   - Add required environment variables

4. **Run the app**
   ```bash
   flutter run
   ```
