### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/foody.git
   cd foody
   ```

2. **Install dependencies**
   ```bash
   npm install
   # or
   yarn install
   ```

3. **Environment Setup**
   ```bash
   cp .env.example .env
   ```

