## üöÄ Getting Started

### Prerequisites

- Flutter SDK (>=2.18.0)
- Dart SDK
- Android Studio / Xcode
- VS Code (recommended)

### Installation

1. **Clone the repository**
   ```bash
   git clone [repository-url]
   cd foody
   ```

2. **Install dependencies**
   ```bash
   flutter pub get
   ```

3. **Configure environment**
   - Create `.env` file in root directory
   - Add required environment variables

4. **Run the app**
   ```bash
   flutter run
   ```
